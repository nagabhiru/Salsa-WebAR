Marker Instructions

  1. Create a high-contrast image with the word SALSA.
  2. Open the AR.js Marker Training tool: AR.js Marker Tool.
  3. Upload your SALSA image and export the .patt file.
  4. Save it as /assets/markers/salsa.patt in this project.
  5. Print the generated marker image for testing.
